Analysis Modes

Analysis buttons

Analyze runs the local SCI402 proposal rubric check. It is fast and does not call the LLM.

AI score asks the LLM to judge rubric quality, then applies local guardrails such as cap rules and evidence checks.

Generate feedback produces tutor-style revision feedback using the selected feedback style.

Feedback style modes

Supportive inquiry is used for blank, very short, or confused drafts. It asks one low-pressure question to help the student start.

Structured guidance is used for incomplete drafts. It gives concrete rubric-based gaps and next revision steps.

Expert challenge is used for stronger drafts. It pushes on limitations, validation, scientific interpretation, and risk mitigation.

Score source

Local rules means scores come from the deterministic rubric assessor. AI-assisted means the LLM judged the draft and the local validator checked the result. Local fallback means AI scoring was unavailable or invalid, so the app kept the local score.
